Evaluation

Overview

Now that we have a trained model, we need to understand how to assess model performance.

Objectives

  • Design own experiments for evaluating a trained model
  • Analyze own computational work and the work of others

Videos

To be replaced with videos:

  2. Recall and precision

Student Activity

  1. In pairs, program one of the following to evaluate the model's performance:
    • Accuracy
    • Precision
    • Confusion Matrix
  2. Design an experiment to evaluate a class of your choosing.
  3. Test your experiment in the real-world, if PortAudio is available on your system.
  4. Discuss your team's results with the rest of the class.
