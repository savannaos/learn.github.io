Setup the codebase:

  1. Clone git repo: `git clone https://github.com/savannaos/devicehive-audio-analysis`
  2. Install dependencies following README
  3. Run the device hive model against different inputs: `python3 parse_file.py path_to_your_file.wav`
Note: PortAudio allows audio capturing which is a fun exercise for testing the model against real-world sounds. It can be disabled if setup becomes too tedious.
