Data Representation

Overview

Now that we have the data, we need to represent it in a way that the machine can perform computations on it.

Objectives

  • Understand how to vectorize audio data
  • Explain how different views of data can tell a different story

Student Activity

  1. Discuss:
    • What does it mean to vectorize words and why do we need to do it?
    • Explain the tradeoffs between one-hot, unique number, and embeddings.
    • How else could you collect sound data? Would it scale?
  2. Select wav files for three different classes
  3. Create spectogram visuals of the waveforms
  4. Disuss the similarities and differences among selected classes
